# Meraki Ads — Meta Live Dashboard

Live Meta Ads Intelligence Dashboard for Meraki Ads agency.

## Setup

1. Get a Meta access token from [Graph API Explorer](https://developers.facebook.com/tools/explorer)
2. Paste it in the token bar on the dashboard, or set `META_ACCESS_TOKEN` env var on Vercel

## Deploy

Deployed on Vercel. Push to `main` to auto-deploy.
